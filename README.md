# sqlite-py
Simple sqlite manager that use CLI prompt from InquirerPy package.  
  
## features  
- Create row(ongoing)
- Update row
- Delete row
- Read Table  
  
## requirements
- Python
- InquirerPy
- Terminal  
  
## installation  
to configure the project,

```bash 
git clone https://github.com/darkx764/sqlite-py.git
python -m venv venv # (optional) to create virtual environment
pip install -r requirements.txt
```  
  
then create a folder name 'databases' in the root path.
  
run the program
```bash
python main.py
```
